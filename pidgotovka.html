Підготовка до встановлення Windows 11

Покрокова інструкція з підготовки вашого комп'ютера до нової операційної системи

Спосіб 1: Як завантажити ISO-файл Windows 11

Крок 1: Перевірка вимог системи

Перед встановленням Windows 11 переконайтеся, що ваш комп'ютер відповідає мінімальним вимогам. Microsoft пропонує застосунок PC Health Check для перевірки сумісності.

  • Процесор: 1 ГГц або швидше з двома або більше ядрами
  • ОЗП: 4 ГБ або більше
  • Сховище: 64 ГБ або більше
  • Системна прошивка: UEFI з підтримкою Secure Boot
  • TPM: TPM 2.0
  • Графічна карта: сумісна з DirectX 12
  • Дисплей: діагональ більше 9 дюймів, роздільна здатність HD (720p)

Крок 2: Вибір мови інсталятора

Виберіть мову, яку ви будете використовувати під час встановлення Windows 11.

  • Виберіть мову зі списку доступних опцій
  • Переконайтеся, що обрана мова відповідає вашим потребам
  • Натисніть кнопку "Confirm" для продовження

Крок 3: Завантаження Windows 11

Завантажте офіційний інсталятор Windows 11 з веб-сайту Microsoft.

  • Виберіть Windows 11 English International
  • Натисніть кнопку "64-bit Download"
  • Зачекайте, поки файл завантажиться на ваш комп'ютер
  • Перевірте цілісність завантаженого файлу

Крок 4: Процес завантаження

Стежте за процесом завантаження файлу інсталяції Windows 11.

  • Зверніть увагу на розмір файлу (приблизно 5.1 ГБ)
  • Не закривайте вікно завантаження до його завершення
  • Переконайтеся, що на диску достатньо місця
Спосіб 2: Створення інсталяційного диска Windows 11 через Media Creation Tool

Крок 5: Створення інсталяційного носія

Використовуйте інструмент Media Creation Tool для створення завантажувального USB-накопичувача.

  • Запустіть завантажений інструмент Media Creation Tool
  • Виберіть опцію "Створити інсталяційний носій"
  • Дотримуйтесь інструкцій для створення USB-накопичувача
  • Переконайтеся, що ваш USB-накопичувач має об'єм не менше 8 ГБ

Крок 6: Налаштування інсталяції

Майстер налаштування Windows 11 підготує необхідні файли для встановлення.

  • Зачекайте, поки система підготує всі необхідні компоненти
  • Не переривайте процес підготовки файлів
  • Перевірте, що комп'ютер підключено до стабільного джерела живлення
  • Користуйтеся кнопками "Back" та "Next" для навігації

Крок 7: Ліцензійна угода Microsoft

Ознайомтеся з ліцензійною угодою Microsoft перед продовженням встановлення.

  • Уважно прочитайте умови ліцензійної угоди
  • Переконайтеся, що розумієте умови використання програмного забезпечення
  • Для продовження натисніть кнопку "Accept"
  • Якщо ви не згодні з умовами, натисніть "Decline" (встановлення буде скасовано)

Крок 8: Вибір мови та редакції Windows

Виберіть мову інтерфейсу та редакцію Windows 11, яку бажаєте встановити.

  • Оберіть мову зі списку доступних варіантів
  • Виберіть редакцію Windows 11, яка відповідає вашій ліцензії
  • Увімкніть опцію "Use the recommended options for this PC" для автоматичного вибору
  • Натисніть "Next" для продовження

Крок 9: Вибір типу інсталяційного носія

Виберіть, який тип носія ви бажаєте використовувати для встановлення Windows 11.

  • Виберіть опцію "USB flash drive" для створення завантажувального USB-накопичувача
  • Виберіть опцію "ISO file" для створення ISO-образу
  • Враховуйте, що для ISO-файлу потрібно буде пізніше записати його на диск чи накопичувач або встановити його як показано у способі 1

Крок 10: Вибір USB-накопичувача

Виберіть USB-накопичувач, який буде використано для створення інсталяційного носія.

  • Переконайтеся, що USB-накопичувач правильно підключено до комп'ютера
  • Виберіть потрібний накопичувач зі списку
  • Зверніть увагу на попередження: всі файли на вибраному накопичувачі будуть видалені

Крок 11: Створення Windows 11 носія

Зачекайте, поки система створить завантажувальний носій Windows 11.

  • Стежте за індикатором прогресу
  • Не переривайте процес створення носія
  • Процес може тривати від 15 хвилин до години, залежно від швидкості інтернету та USB-накопичувача

Крок 12: Завершення створення носія

Завантажувальний USB-накопичувач Windows 11 готовий до використання.

  • Після появи повідомлення "Your USB flash drive is ready" процес завершено
  • Натисніть "Finish" для завершення
  • Тепер ви можете використовувати цей накопичувач для встановлення Windows 11
  • Перезавантажте комп'ютер і виберіть завантаження з USB для початку встановлення
Спосіб 3: Неофіційний метод через Rufus

Крок 1: Налаштування Rufus

Rufus - це утиліта для створення завантажувальних USB-накопичувачів, що дає більше опцій контролю та може обійти деякі обмеження встановлення Windows 11.

  • 1) Оберіть USB-накопичувач, який хочете використати для інсталяції. Переконайтеся, що вибрали правильний пристрій, оскільки всі дані на ньому будуть видалені
  • 2) Виберіть ISO-образ операційної системи, яку бажаєте встановити (для Windows 11 виберіть завантажений ISO-файл)
  • 3) Якщо у вас немає ISO-файлу, натисніть кнопку "SELECT" для його вибору
  • 4) Виберіть схему розділів (для Windows 11 рекомендується GPT для UEFI). MBR - для старіших комп'ютерів з BIOS, GPT - для сучасних комп'ютерів з UEFI. Якщо ваш комп'ютер випущений після 2012 року, скоріше за все вам потрібен GPT
  • 5) Введіть мітку тому (за бажанням) або залиште запропоновану назву - це буде ім'я вашого USB-накопичувача після форматування
  • 6) Виберіть файлову систему. Для Windows 11 рекомендується NTFS, але FAT32 підходить для більшості випадків, крім файлів розміром понад 4 ГБ
  • 7) Виберіть розмір кластера. Зазвичай можна залишити значення за замовчуванням "4096 bytes (Default)"
  • 8) Натисніть кнопку "START", щоб розпочати процес створення завантажувального носія

Крок 2: Процес створення носія

Після натискання кнопки "START" почнеться процес створення завантажувального USB-накопичувача.

  • Погодьтеся з попередженням про видалення всіх даних на USB-накопичувачі
  • Rufus почне форматування накопичувача та копіювання файлів
  • На екрані буде відображатися прогрес створення носія (як показано на зображенні)
  • Не відключайте USB-накопичувач і не вимикайте комп'ютер під час цього процесу

Крок 3: Завершення та попередження про Secure Boot

Після завершення процесу створення Rufus покаже важливе повідомлення.

  • Зверніть увагу на попередження: для завантаження з цього носія потрібно вимкнути Secure Boot у налаштуваннях BIOS/UEFI
  • Це обмеження виникає через те, що Rufus створює носій, який обходить деякі обмеження Windows 11
  • Натисніть "Додаткова інформація", щоб дізнатися більше, або "Close", щоб закрити вікно
  • Тепер ваш USB-накопичувач готовий до використання для встановлення Windows 11
